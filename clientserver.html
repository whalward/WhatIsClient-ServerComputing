Client/Server Computing

by Waseem Alward

What is the client server model?

The client server model is a computer/sofware/network model where some computers/software make requests for data and other computers/software listen for and respond to these requests (servers).

Advantages of the Client/Server Model

  1. Scalability
  2. Security and Administration
  3. Client/User Backup and Recovery
  4. Shared Data

Examples of TCP/IP Protocols

  • FTP - Protocol used to send/transfer/recieve files from one computer to another
  • HTTP - Protocol used to send/retrieve web pages over the Web using a Web browser
  • HTTPS - Protocol used to send/retrieve web pages securely/encrypted over the Web using a Web broswer
  • DHCP - Protocol used to dynamically assign IP addresses to the deviced on the Internet
  • SMTP - used to send email messages

Examples of network utilities and services

  • ping - utility used to test whether the host on a network is reachable/availiable
  • tracert - utility that displays the path packets are taking to get from your current location/device on a network to another location/network. The number of hops and how long each hop takes is displayed allowing one to see if and where there are any delays between two locations.
  • pathping- utility that combines features of ping and tracert
  • whois - utility that allows you to look-up who owns and Internet resource, such as a domain name or IP address block.
  • ipconfig - utility that allows you to look-up and configure a devices IP address, DHCP, and DNS information
